# 养老管理系统 java992

## 项目概述

养老管理系统是基于 Spring Boot 和 Vue.js 技术栈构建的一套前后端分离的应用程序，旨在提供老年人档案管理、护理服务调度、健康监测、活动安排及家属互动等全方位服务。

## 技术栈

### 后端技术

- Spring Boot
- MyBatis
- MySQL
- Redis
- Spring Security
- Spring Cloud

### 前端技术

- Vue.js
- Element UI

### 开发工具

- IntelliJ IDEA
- Maven

## 功能模块

- 老年人档案管理
- 护理服务调度
- 健康监测
- 活动安排
- 家属互动

## 系统角色

系统主要包含以下角色：

- 管理员
- 护理工作人员
- 家属
- 老年人

## 运行环境

- 服务器：推荐使用 Linux 或 Unix 系统
- 数据库：MySQL
- 缓存：Redis
- 前端：现代浏览器

## 部署步骤

1. 准备运行环境（数据库、缓存等）
2. 部署后端服务（Spring Boot 应用）
3. 部署前端服务（Vue.js 应用）
4. 配置域名及反向代理（可选）

- 基于成熟的 Spring Boot 和 Vue.js 技术栈，确保系统的稳定性和可扩展性
- 提供全面的功能模块，满足养老管理的多样化需求
